* Weight Manager

** Overview
Easy weight management application. Manage daily weight changes while
looking at graph. Record the date and the weight of the day.

Technology

- front_end
  - React
  - material-UI

- back_end
  - firebase

** Warning
When editing with emacs

Please edit.

- before
#+begin_src 
watchOptions: {
ignored: ignoredFiles(paths.appSrc),
},	
#+end_src

- after
#+begin_src 
watchOptions: {
ignored: [ignoredFiles(paths.appSrc), "**/.#*", "**/*~", "**/#*#"],
},
#+end_src
